# 외국인 에스테틱 예약 앱 — 개발자 전달용 화면 구성안

> 가칭: kbeauty-book / glow-trip / aesthe-link
>
> **목표**: 외국인 관광객이 한국 에스테틱 샵을 찾고, 예약하고, 카드로 결제하고, 요청사항을 번역해서 전달할 수 있는 앱

---

## 1. 꼭 필요한 화면 (MVP)

| 구분 | 화면 |
|------|------|
| 고객용 | 시작 화면 / 로그인·회원가입 / 홈 / 샵 목록 / 샵 상세 / 예약 / 결제 / 예약 완료 / 예약 내역 / 마이페이지 |
| 원장님용 | 원장님 로그인 / 예약 관리 / 예약 상세 / 스케줄 관리 |
| 관리자용 | 관리자 로그인 / 대시보드 / 샵 관리 / 정산 관리 |

---

## 2. 화면별 핵심 내용

- **시작 화면**: 로고, 한 줄 소개, 로그인 버튼, 회원가입 버튼, 언어 선택
- **홈 화면**: 검색창, 지역 선택, 카테고리, 추천 샵, 하단 메뉴
- **샵 목록**: 사진, 샵 이름, 별점, 가격, 정렬·필터, 예약 버튼
- **샵 상세**: 샵 사진, 주소, 지도, 운영시간, 시술 목록, 가격, 리뷰, 예약 버튼
- **예약 화면**: 날짜 선택, 시간 선택, 요청사항 입력, 총 금액, 다음 버튼
- **결제 화면**: 예약 정보, 카드 결제 버튼, 취소/환불 규정
- **원장님 관리**: 예약 목록, 상태 변경, 요청사항 원문+번역본, 스케줄 열기/닫기
- **관리자 화면**: 전체 예약 수, 입점 샵 수, 결제 금액, 정산 금액

---

## 3. 원하는 앱 느낌

- 깔끔하고 보기 쉬운 느낌
- 외국인이 써도 복잡하지 않은 구조
- 에스테틱 서비스답게 세련되고 믿음 가는 분위기
- 너무 화려하지 않고, 화이트/베이지/연한 블루·그린 계열 선호

---

## 4. 개발자와 바로 맞춰볼 것

- 이 문서가 와이어프레임 수준인지, 실제 UI 시안 수준인지 확인
- 처음 버전에서 꼭 필요한 화면과 나중에 넣을 화면 구분
- 합칠 수 있는 화면이 있는지 확인
- 디자인 포함 범위와 수정 횟수 확인
